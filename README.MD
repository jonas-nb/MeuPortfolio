## Instalação ⚙

Instale AdviceApp com npm

```bash
  git clone https://github.com/jonas-nb/AdviceApp
  cd AdviceApp
  npm install
  npm start
```

<br/>

## Deploy ☁

Para fazer o deploy desse projeto rode

```bash
  npm run build
```
